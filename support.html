Support

Extension help and issue reporting.

For bugs, feature requests, and usage questions, use the GitHub issue tracker linked from the relevant extension page. Include your VS Code version, extension version, operating system, and clear reproduction steps when possible.

Security

If you need to report a security concern, do not post sensitive details in a public issue. Use the private security reporting channel listed in the extension repository, or contact the maintainer directly.

Response times

Crunchy Codes projects are independently maintained. Support requests are reviewed as availability allows, with priority given to bugs, regressions, and security-related reports.
